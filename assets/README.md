# Assets
This folder contains visual and supporting materials related to the FP&A Financial Dashboard project.

## Files Included
### dashboard_preview.png
Static image of the final Tableau dashboard layout.
Used for quick visual reference within the main repository README.

### FPnA_Financial_Performance_Walkthrough.mov

Short screen recording demonstrating:

- Net Margin trend visualization
- Revenue trend from the year 2023 - 2025 
- Dashboard interaction
- KPI interpretation
- Cost trend analysis
- Revenue by category
- Profit trend 

The walkthrough provides a concise overview of how financial performance evolves across reporting periods.

## Purpose

The assets folder supports:

- Visual documentation
- Portfolio presentation
- Executive-style demonstration of analytical output
